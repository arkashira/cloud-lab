- **Email Notification** – If an `email` address is supplied, the system sends an email containing the share URL via the `mailer` service.  
- **Response Payload** – The API returns a JSON object `{ url: <share‑link>, token: <token> }`.  
- **Retrieval** – `getSharedEnvironment` returns environment details (excluding sensitive fields) when the token is valid and not expired; otherwise it returns `404 Not Found`.  
- **Acceptance** – `acceptSharedEnvironment` requires authentication, ensures the user is not the owner, verifies the token is valid and not already accepted, updates `accepted_at`, and creates a new environment record for the recipient with the specified `permission`.
