This repository contains a collection of pre-optimized Docker images designed to accelerate ML API deployments by reducing image size and improving startup times.

## Available Images

The following images are available for the most common ML frameworks:

### 1. TensorFlow
- **Image**: `axentx/cloud-lab-tensorflow:latest`
- **Features**:
  - Optimized for TensorFlow 2.x
  - Includes popular pre-trained models (ResNet, BERT)
  - Minimal runtime dependencies
  - Size: ~1.2 GB
- **Usage**:
  ```bash
  docker pull axentx/cloud-lab-tensorflow:latest
  docker run -p 5000:5000 axentx/cloud-lab-tensorflow:latest
  ```

### 2. PyTorch
- **Image**: `axentx/cloud-lab-pytorch:latest`
- **Features**:
  - PyTorch 1.13 optimized build
  - CUDA 11.6 support
  - Pre-installed torchvision and torchaudio
  - Size: ~1.1 GB
- **Usage**:
  ```bash
  docker pull axentx/cloud-lab-pytorch:latest
  docker run -it --rm -v $(pwd):/app axentx/cloud-lab-pytorch:latest python app.py
  ```

### 3. Scikit-learn
- **Image**: `axentx/cloud-lab-scikit:latest`
- **Features**:
  - Scikit-learn 1.2 with optimized algorithms
  - Pandas and NumPy included
  - LightGBM and XGBoost dependencies
  - Size: ~600 MB
- **Usage**:
  ```bash
  docker pull axentx/cloud-lab-scikit:latest
  docker run -it axentx/cloud-lab-scikit:latest python -c "from sklearn.datasets import load_iris; print(load_iris().data)"
  ```

### 4. FastAPI + TorchServe
- **Image**: `axentx/cloud-lab-fastapi-torchserve:latest`
- **Features**:
  - FastAPI framework with TorchServe integration
  - Production-ready serving environment
  - gunicorn and Nginx pre-configured
  - Size: ~1.5 GB
- **Usage**:
  ```bash
  docker pull axentx/cloud-lab-fastapi-torchserve:latest
  docker run -p 80:80 axentx/cloud-lab-fastapi-torchserve:latest
  ```

## How to Contribute

1. **Build a new image**:
   ```bash
   docker build -t axentx/cloud-lab-<framework>:<tag> .
   ```

2. **Push to repository**:
   ```bash
   docker push axentx/cloud-lab-<framework>:<tag>
   ```

3. **Update README** with new image details.

## Documentation

For detailed usage instructions and API references, refer to the [cloud-lab documentation](https://cloud-lab-docs.example.com/).
