### Usage Instructions

To use the pre-optimized Docker images, follow these steps:

1. **Pull the Image**:
   ```bash
   docker pull axentx/<framework>:latest
   ```
   or
   ```bash
   docker pull axentx/ml-<framework>
   ```
2. **Run the Container**:
   ```bash
   docker run -it axentx/<framework>:latest
   ```
   or
   ```bash
   docker run -it axentx/ml-<framework>
   ```
3. **Customize the Image**:
   You can create a custom Dockerfile that extends one of the pre-optimized images:
   ```dockerfile
   FROM axentx/<framework>:latest
   COPY . /app
   WORKDIR /app
   RUN pip install -r requirements.txt
   CMD ["python", "app.py"]
   ```
