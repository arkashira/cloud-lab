## Introduction
This Terraform template provisions a basic AWS environment for practicing infrastructure as code.

## Prerequisites
* Terraform installed on your machine
* AWS account with necessary credentials (IAM user with programmatic access)

## Setup Instructions
1. Clone the repository: `git clone https://github.com/[username]/cloud-lab.git`
2. Navigate to the template directory: `cd /opt/axentx/cloud-lab/templates/terraform`
3. Initialize Terraform: `terraform init`
4. Validate the configuration: `terraform plan`
5. Apply the changes: `terraform apply`

## Resources
* AWS VPC (Virtual Private Cloud)
* AWS Subnet (public/private)
* AWS Security Group (with SSH/HTTP rules)
* AWS EC2 Instance (Ubuntu Linux)

## Configuration File
* Update `terraform.tfvars` with your AWS credentials and region:
  ```
  aws_region = "us-east-1"
  aws_access_key = "YOUR_ACCESS_KEY"
  aws_secret_key = "YOUR_SECRET_KEY"
  ```

## Notes
* This template is for educational purposes only and should not be used in production without proper testing and validation.
* Ensure you have appropriate IAM permissions (e.g., `ec2:*, vpc:*`)
* The security group allows SSH access from your IP and HTTP traffic to port 80
