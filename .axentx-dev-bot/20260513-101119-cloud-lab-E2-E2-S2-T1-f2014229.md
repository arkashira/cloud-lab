| What we improved | Why it matters |
|------------------|----------------|
| **Explicit `become: true`** on every task | Guarantees the role works on a fresh EC2 instance where the default user (e.g. `ec2‑user`) does not have root privileges. |
| **`ansible.builtin.package`** (instead of the generic `package`) | Makes the module name explicit for readers and for static‑analysis tools. |
| **`block / rescue`** pattern for each logical step | Gives a single, clean failure message while still letting Ansible continue to the next block when you want it to (e.g. you may want the log even if the S3 upload fails). |
| **`failed_when` / `changed_when`** instead of `ignore_errors + when: … is failed` | Removes the “double‑negative” pattern and makes the intent clearer. |
| **Variables with defaults in `defaults/main.yml`** | Centralises configuration (bucket name, region, key path, etc.) and makes the role reusable across environments. |
| **`community.aws.s3`** (the modern S3 module) | The older `aws_s3` is still supported but the community collection is the recommended, actively‑maintained implementation. |
| **Idempotent log‑file naming** (`strftime` with seconds) | Guarantees a unique filename *only* when the task actually runs (i.e. when something changed). |
| **`notify` → handler** to print a success banner only once | Keeps the play output tidy. |
| **Extensive inline documentation** | Makes the role easy to hand‑off to other team members. |

from ansible.parsing.dataloader import DataLoader
from ansible.vars.manager import VariableManager
from ansible.playbook.play import Play

# Helper to run a single role task with given vars
def run_role_task(task_name, extra_vars):
    loader = DataLoader()
    variable_manager = VariableManager(loader=loader, inventory=None)
    variable_manager.extra_vars = extra_vars
    # Build a minimal play that includes only the target task
    play_source = dict(
        name="Validate vars",
        hosts="localhost",
        gather_facts=False,
        roles=[dict(role="setup-ec2", tasks_from=task_name)],
    )
    play = Play().load(play_source, variable_manager=variable_manager, loader=loader)
    tqm = TaskQueueManager(
        inventory=None,
        variable_manager=variable_manager,
        loader=loader,
        passwords={},
    )
    result = tqm.run(play)
    return result
