<!-- tutorials/azure_nodejs.md -->
# Tutorial: Deploy a Node.js Web App to Azure

Welcome to this step-by-step tutorial where you'll deploy a Node.js web application to Azure using the Azure CLI.

## Prerequisites
- An Azure account (free trial at azure.com)
- Node.js installed locally (v14 or higher)
- Git installed

---

## Step 1: Login to Azure

Open your terminal and run the following command to authenticate with Azure:

**Command to execute:**
```
az login
```

---

## Step 2: Create a Resource Group

A resource group is a logical container for Azure resources:

**Command to execute:**
```
az group create --name myNodeAppRG --location eastus
```

---

## Step 3: Create an App Service Plan

An App Service plan defines the compute resources for your web app:

**Command to execute:**
```
az appservice plan create --name myNodeAppPlan --resource-group myNodeAppRG --sku FREE
```

---

## Step 4: Create the Web App

Now let's create the web app in Azure:

**Command to execute:**
```
az webapp create --resource-group myNodeAppRG --plan myNodeAppPlan --name myNodeApp-$(date +%s)
```

> Note: The app name must be globally unique.

---

## Step 5: Deploy Your Code

Deploy your Node.js app from a local Git repository:

**Command to execute:**
```
az webapp deployment source config-local-git --name <your-app-name> --resource-group myNodeAppRG
```

---

## 🎉 Congratulations!

Your Node.js app is now deployed to Azure. 

**View your live app:** `https://<your-app-name>.azurewebsites.net`
```

---
