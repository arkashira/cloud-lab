# Marketing Plan

## Positioning

**1-line positioning:**  
"Learn AWS hands-on without spending a penny—cloud-lab gives developers unlimited sandbox access for free."

**3 Alternatives:**  
1. "Practice AWS skills in a risk-free sandbox environment, no credit card required."  
2. "Free AWS lab environment for developers who want to learn without financial risk."  
3. "AWS training without cost—practice real cloud infrastructure in a sandbox."

## ICP

**1. Sarah, Junior DevOps Engineer**  
- Daily job: Learning AWS deployment patterns, CI/CD pipelines, and cloud security best practices.  
- Budget: $0/month (free tier only).  

**2. Mike, Full Stack Developer**  
- Daily job: Building web apps using AWS Lambda, S3, and DynamoDB; needs sandbox to test ideas.  
- Budget: $0–$50/month (personal projects).  

**3. Priya, Engineering Manager**  
- Daily job: Evaluating AWS tools for team training and onboarding new hires.  
- Budget: $0–$100/month (team learning budget).  

## Channels

**1. GitHub Community + DevRel Partnerships**  
- CAC: ~$10  
- Strategy: Leverage existing open-source community, partner with AWS educators and dev advocates.  

**2. Dev.to + Hashnode Content Syndication**  
- CAC: ~$20  
- Strategy: Publish tutorials and guides that link back to cloud-lab.  

**3. Reddit (r/aws, r/devops, r/learnprogramming)**  
- CAC: ~$15  
- Strategy: Share beginner-friendly AWS labs and engage in discussions around learning platforms.  

## Content Cadence

| Week | Focus Area | Content Type |
|------|------------|--------------|
| Week 1 | Awareness | Launch announcement + intro video |
| Week 2 | Education | AWS basics tutorial series |
| Week 3 | Engagement | Community Q&A + live demo |
| Week 4 | Conversion | Free trial sign-up + testimonials |
| Week 5 | Retention | Weekly challenge + progress tracking |
| Week 6 | Growth | Referral program launch |
| Week 7 | Expansion | Advanced lab walkthrough |
| Week 8 | Feedback Loop | User survey + feature request campaign |

## Launch Milestones

| Date | Milestone |
|------|-----------|
| D-30 | Finalize pricing tiers ($0/month for free tier, $5/user/mo for premium features) |
| D-15 | Pre-launch content rollout (dev.to, Reddit, GitHub) |
| D-0 | Public launch + social media blitz |
| D+30 | First MRR milestone: $1,000/month from paid users |
| D+90 | Scale to $5,000/month MRR with 100+ paying users |

## Success Metrics

| Metric | Target by D+90 |
|--------|----------------|
| DAU | 1,000 active users |
| MRR | $5,000/month |
| Paid Users | 100+ paying users |
| Conversion Rate | 2% from free to paid |
| Net Promoter Score (NPS) | >60 |
